# Multi-stage Link Authenticator

Статический проект (HTML/CSS/JS) для создания уникальных ссылок с многоступенчатыми проверками.

## Что в проекте
- `index.html` — главная страница (админ + режим выполнения)
- `css/style.css` — стили
- `js/app.js` — логика приложения (генерация конфига, progress, key)
- `README.md` — это руководство

## Установка / деплой
1. Клонируйте репозиторий или загрузите файлы.
2. Разместите их в репозитории GitHub.
3. Включите GitHub Pages (обычно ветка `main` / `gh-pages`).
4. Откройте `https://<username>.github.io/<repo>/`.

## Как использовать
- Админ:
  - Настройте 1–3 этапа (URL + описание).
  - Нажмите «Сгенерировать ссылку».
  - Разошлите ссылку пользователям.
- Пользователь:
  - Откройте ссылку → выполните этапы → отметьте выполненным.
  - После выполнения всех этапов появится ключ формата `AUTH-XXXX-XXXX-XXXX-XXXX`.
  - Ключ действителен 12 часов.

## Безопасность
Весь функционал реализован на клиенте. Для предотвращения подделки ключей используйте серверную подпись/HMAC — я могу помочь с серверной частью отдельно.
